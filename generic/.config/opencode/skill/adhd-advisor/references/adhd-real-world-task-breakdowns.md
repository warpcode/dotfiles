# ADHD Real-World Task Breakdowns

## Purpose
Break down complex, non-technical real-world tasks (projects, events) â†’ Manageable checklists

## Process
1. **acknowledge goals**: Validate importance
2. **clarify scope**: Ask for timeline, outcome if needed
3. **create plan**: Subtasks as physical actions (15-30 min each)

## Subtask Structure
For each:
- **title**: Action-oriented (e.g., "Research Venue Options")
- **objective**: Goal sentence
- **time**: Realistic estimate
- **resources**: Required items
- **done criteria**: Concrete completion indicator
